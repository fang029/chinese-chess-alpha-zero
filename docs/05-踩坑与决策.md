# 踩坑与决策

记录"为什么这么做"和"踩过的坑",避免重蹈覆辙。

## 关键决策

### D1. 加速靠 CPU 核数,不是 GPU
AlphaZero 瓶颈是自我对弈(占 90%+ 时间),它是 batch=1 小推理,GPU 喂不饱。
实测满规模网络单局 GPU 每步 2490ms vs CPU 4456ms,GPU 只快 1.8 倍,而 A10
租金是同核 CPU 数倍。**结论**:要快加 CPU 核;堆 GPU/换强卡是浪费。唯一能让
GPU 真正有用的是改造"批量推理服务"(把 worker 的小推理凑大 batch),几百行
工程 + 调试风险,暂未做。用户计划换 64 核纯 CPU 跑自我对弈。

### D2. 用小网络 64×6,不用满规模 128×10
满规模在 CPU 自我对弈单局分钟级,反馈周期过长。64×6 推理快 3-4 倍,先练出
基础棋力,之后可加大网络续训。**注意**:续训时 channels/blocks 改了会
load_state_dict size mismatch,只能从头练。

### D3. 先预训练破冷启动
纯随机冷启动训练 181 轮,loss 1.25 触底后回升到 1.5、几乎全和棋——网络学不出
棋力。改用 14 万局人类棋谱监督预训练(见 docs/04)。

### D4. 预训练用 GPU、自我对弈用 CPU
预训练是大 batch 密集计算(GPU 擅长),一次性;自我对弈是 batch=1(CPU 划算)。
建议带 GPU 的机器先把预训练跑完存 pretrained.pt,再换纯 CPU 跑自我对弈。

### D5. 并行编码 + 磁盘分片,不在训练时实时编码
单进程编码全量 7 小时/epoch 且每 epoch 重编码。改为 30 进程一次性编码到磁盘
分片(~40 分钟),训练直接读分片多 epoch 复用。1160 万样本全量编码 ~230GB,
为此把磁盘从 99G 扩到 1TB。

## 踩过的坑

### K1. 系统 Python 3.6 装不了 PyTorch
云服务器镜像自带 py3.6.8。用 conda 建 py3.10 环境 `xiangqi`。conda.sh 在
`/etc/profile.d/`(非 `conda info --base` 返回的 /usr),脚本里**直接用 env
绝对路径 python** `python`,别用 conda activate。

### K2. pip 官方源在国内慢到不可用
torch 下载 ~12KB/s。换清华镜像 `-i https://pypi.tuna.tsinghua.edu.cn/simple`
提到 ~10MB/s。

### K3. 服务器缺 rsync/git/screen
rsync 没有 → 部署用 tar over ssh。git/screen 用 `dnf install -y` 装。

### K4. Trainer 没把 net 搬到 device(真 bug,已修)
`Trainer.__init__` 之前没 `self.net.to(device)`,GPU 训练报 input/weight 设备
不匹配。只有真上 GPU 才暴露(CPU 跑不出来)。已修复。

### K5. train.log 经 tee 管道块缓冲,看着像卡死
Python stdout 经管道变块缓冲,日志长时间不刷新,CPU 却满载。判断真卡死要看
进程 CPU 时间 / metrics.csv / checkpoints 文件时间,不是看 log。已给训练命令
加 `-u`(无缓冲)缓解。**看进度一律用 metrics.csv**。

### K6. screen/SSH 在高负载下返回 255
CPU 负载满(如 28/32)时 SSH 可能拒连或命令返回 255。kill 命令往往已生效,
重连确认即可(`ps aux | grep -c`)。

### K7. pipeline_parallel 不支持 --gating
评估门控只在串行 `pipeline.py`。给并行管线传 `--gating` 会 argparse 报错。

### K8. encode_dataset 中途 kill 留损坏分片
最后一个分片写一半被 kill,np.load 会报错。训练前删掉最后一个不完整分片。

### K9. pretrain_from_shards 定义位置
必须定义在 `if __name__=='__main__'` 之前,否则 main 调用时 NameError。
(append 代码到文件末尾时容易踩。)

### K10. 单核编码瓶颈
编码循环原本单进程,32 核只用 1 核。改 `encode_dataset.py` 多进程 Pool,
快 ~25 倍。
